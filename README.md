CoDir Server
Instructions:
1: run "node index.js run [dir] -p [port]"
2: connect with CoDir Sublime

Codir Sublime
Instructions:
1: move /codirSublime in Sublime Text 3's /Packages directory
2: click "Tools->Connect To..."
3: enter [your ip]:[port] in the input field and press Enter
